# STA220-NBA-Project
The finalizedcenters notebook was used to retrieve data and create our SQL database. While the code runs, you will most likely be rate limited and the process takes several hours. Therefore, we stongly recommend running only the finalvisualization and results notebooks, as the data_cleaning notebook connects to the database created in finalized centers and outputs the CSVs used in our analysis. finalvisualization creates the visualizations used in the presentation, while the results is where we conducted our statistical analysis.

In order to run these, you will need the data collected. As the file size is too large to upload to github, you can download the required CSVs from my [Google Drive](https://drive.google.com/drive/folders/18JzAf7AYVvYHaAYpEaz5YAMYdgHi9bVm?usp=drive_link). Place both in the same directory you downloaded the python notebooks.
