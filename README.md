# STA220-NBA-Project
The finalizedcenters notebook was used to retrieve data and create our SQL database. While the code runs, you will most likely be rate limited and the process takes several hours. Therefore, we stongly recommend running only the finalvisualization and results notebooks, as the data_cleaning notebook connects to the database created in finalized centers and outputs the CSVs used in our analysis. finalvisualization.ipynb creates the visualizations used in the presentation, while results.ipynb is where we conducted our statistical analysis.

In order to run these, you will need the data collected. As the file size is too large to upload to github, you can download the required CSVs from my [Google Drive](https://drive.google.com/drive/folders/18JzAf7AYVvYHaAYpEaz5YAMYdgHi9bVm?usp=drive_link). Place both in the same directory you downloaded the python notebooks. 

Here are the recommended step by step instructions:
1. Download final_loc and center_loc csvs from my [Google Drive](https://drive.google.com/drive/folders/18JzAf7AYVvYHaAYpEaz5YAMYdgHi9bVm?usp=drive_link).
2. Download the finalvisualization.ipynb and results.ipynb notebooks. Place them in the same folder as the CSVs.
3. Run the downloaded notebooks in whichever order is preferred.
